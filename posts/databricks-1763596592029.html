Databricks

11/19/2025, 6:56:32 PM

  • Databricks is a multi cloud lakehouse framework built upon spark architecture.
  • Main components are Data plan{cloud storage(Data lake) + VM`s on the cloud} and control plan{Web UI, Cluster Management, Workflows, notebooks}
  • created by pyspark
  • DBFS: whenever a cluster is created then it automatically attaches a run time and creates DBFS(databricks file system). It is like an abstract layer of storage created on the cloud service.
  • A Cluster is a group of nodes/computers working together as a one unit.
  • Master node/ Driver Node/ Worker Node
