Databricks: Delta Lake

11/19/2025, 8:19:30 PM

It is an opensource storage framework which brings reliability to data lakes. It enables in building a lake house.

Metadata is stored in a transaction log file which enables the properties of having ACID transactions, no duplicates, versioning, auditing.

It stores the data in the form of parquet files and a delta log file which has the transaction logs in the form of json.

when a delta lake table is created in hive_metastore it stores the file in the dbfs:// location which can be viewed by using fs command.

use catalog hive_metastore

create table default.employee(
int id,
name string,
age int,
salary double)

describe detail employee

insert into employee values(5)
